# mukbanglog

기본아이디어
---
음식을 먹었던 장소를 구글맵에서 보여준다.

입력데이터
---
- 구글맵에서 가게검색
- 방문일시
- 방문목적
  - 친구들과의 모임 / 연인과의 데이트 / 회사 동료와의 모임 / 거래처 사람과의 모임 
- 별점
- 코멘트
- 먹은음식/코스

주요 기능
---
로그인(SNS LOGIN)
 - 페이스북 로그인

먹방지도
 - 자신이 지정한 맵에서 먹었던 곳의 핀을 표시해서 보여줌
 - 특정조건필터해서 맵에서 표시가능 (별 3개이상 / 30일 이내에 방문한 곳 / )

다국어대응
 - 일본어
 - 한국어
 - 영어

# 사용하는 유저가 늘어나면 추천엔진을 작성하는게 가능
